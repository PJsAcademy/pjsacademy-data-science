## 1. 🤖 Multi-Agent Research Assistant
**Difficulty:** ⭐⭐⭐⭐⭐ | **Skills:** LangGraph, agent orchestration, tool use, planning

A team of AI agents that research a topic together — a planner, researchers, a critic, and a writer — producing a cited report.

- **v1:** Single agent with web search
- **v2:** Multi-agent with roles (planner + researcher + writer)
- **v3:** Full crew with critic loop, citation verification, and human-in-the-loop
- **Innovation:** Self-critiquing agent loop that fact-checks itself
- **Resume line:** *"Built a multi-agent research system with LangGraph featuring self-critique and citation verification."*

## 2. 📚 RAG Over Your Entire Life (Personal AI)
**Difficulty:** ⭐⭐⭐⭐ | **Skills:** RAG, vector DBs, multimodal, privacy

A private AI that knows everything you've saved — notes, PDFs, emails, bookmarks — and answers questions with sources, running locally.

- **v1:** RAG over documents
- **v2:** Multi-source (notes + PDFs + web clips) + hybrid search
- **v3:** Local LLM (Ollama) + multimodal (images) + temporal reasoning
- **Innovation:** Fully local, private "second brain" with multimodal recall
- **Resume line:** *"Built a private, local multimodal RAG assistant over personal documents with hybrid search."*

## 3. 🎙️ Real-Time Voice AI Agent

## 6. 🏦 Fine-Tuned Domain Expert LLM
**Difficulty:** ⭐⭐⭐⭐⭐ | **Skills:** LoRA/QLoRA, dataset creation, evaluation, alignment

## 7. 🔍 Semantic Search Engine for Code
**Difficulty:** ⭐⭐⭐⭐ | **Skills:** code embeddings, vector search, AST, ranking

Search a codebase by *meaning*, not keywords — "find where we handle payment retries" returns the right function even without those words.

- **v1:** Embed + search code snippets
- **v2:** AST-aware chunking + hybrid (semantic + keyword) search
- **v3:** Natural-language → code + cross-repo search + explanation
- **Innovation:** AST-aware semantic chunking of code
- **Resume line:** *"Built a semantic code-search engine with AST-aware chunking and hybrid ranking."*
